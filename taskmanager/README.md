# TaskMaster - Complete Task Manager

A professional task management application built with HTML, CSS, JavaScript, and Supabase.

## Features
- User authentication (Sign up / Login)
- Create, read, update, delete tasks
- Task status tracking (Pending / Completed)
- Real-time statistics
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Supabase (Backend + Database)
